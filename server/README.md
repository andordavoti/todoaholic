# todoaholic server

## Getting Started

The project requires [Node.js](https://nodejs.org/), [Yarn](https://yarnpkg.com/), [TypeScript](https://www.typescriptlang.org/) and the [Firebase CLI](https://firebase.google.com/docs/cli).

To run the app, open the `index.ts` file in the `functions/src` directory and install the dependecies with:

```sh
yarn
```

Then run the following command:

```sh
yarn serve
```

## Deployment

To deploy the app to Firebase, log into the Firebase CLI, go into the `functions/src` directory and run the following command:

```sh
yarn deploy
```
